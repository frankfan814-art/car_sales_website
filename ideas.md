# 汽车销售官网 - 设计思路

## 设计方向选择

经过思考，我为您的汽车销售官网选择了以下设计方向：

### 选定方向：现代专业风格（Modern Professional）

**设计理念**：展现汽车行业的专业性、信任感和现代感。通过精心的排版、充足的留白和优雅的交互，打造高端、可信的品牌形象。

#### 核心原则
1. **信任与专业**：深色背景配合高对比度文字，传递稳定和专业的品牌形象
2. **视觉层级清晰**：通过大小、颜色、间距的差异化处理，引导用户关注重要信息
3. **现代简约**：避免过度装饰，用留白和几何形状营造现代感
4. **易用性优先**：导航清晰，信息架构合理，用户能快速找到想要的内容

#### 色彩哲学
- **主色**：深蓝色 (#1F2937) - 代表专业、信任和稳定
- **辅助色**：金色/琥珀色 (#F59E0B) - 用于强调重要信息和CTA按钮，增加高端感
- **背景**：浅灰白 (#F9FAFB) - 清爽、专业的背景
- **文字**：深灰/黑色 (#111827) - 高对比度，易读性强

#### 布局范式
- **顶部导航**：固定式导航栏，品牌logo + 菜单项 + 联系按钮
- **Hero区域**：大尺寸标题 + 副标题 + 背景图片（汽车相关）+ CTA按钮
- **车型展示**：网格布局，每个车型卡片包含图片、名称、价格、简介
- **特色功能区**：分段式布局，交替左右排列，配合图片和文字
- **底部**：联系信息、社交媒体链接、版权信息

#### 标志性元素
1. **卡片设计**：带阴影的白色卡片，用于展示车型和功能
2. **金色强调线**：在关键区域使用金色分割线或边框，增加高端感
3. **渐变效果**：温和的渐变背景，增加视觉深度

#### 交互哲学
- **悬停效果**：卡片轻微上升、阴影加深，表示可交互
- **按钮反馈**：点击时颜色变化，给予即时反馈
- **平滑过渡**：所有动画使用 0.3-0.5s 的缓动，显得高级而不突兀

#### 动画指南
- **页面加载**：淡入效果，从上到下逐个元素出现
- **卡片悬停**：轻微上移 (4px) + 阴影加深，持续 0.3s
- **按钮点击**：缩放 0.95x，持续 0.1s，然后恢复
- **滚动动画**：使用视差效果，背景图片比前景内容移动更慢

#### 排版系统
- **标题字体**：Playfair Display (serif) - 优雅、高端、专业
- **正文字体**：Lato (sans-serif) - 清晰、易读、现代
- **标题层级**：
  - H1: 48px, 600 weight, 深蓝色
  - H2: 36px, 600 weight, 深蓝色
  - H3: 24px, 600 weight, 深灰色
  - 正文: 16px, 400 weight, 深灰色

## 页面结构规划

### 首页 (Home)
1. **导航栏** - 固定顶部，包含logo、菜单、联系按钮
2. **Hero区** - 大背景图 + 标题 + 副标题 + CTA
3. **热门车型** - 网格展示 3-4 款热门车
4. **为什么选择我们** - 3 个特色优势
5. **最新优惠** - 当前促销信息
6. **客户评价** - 简短的客户反馈
7. **联系我们** - 电话、地址、表单
8. **页脚** - 链接、社交媒体、版权

## 技术实现细节

- 使用 Tailwind CSS 4 + shadcn/ui 组件库
- 响应式设计，支持移动端、平板、桌面
- 使用 Framer Motion 实现平滑动画
- 图片使用本地存储或外部CDN
- 无后端依赖，纯前端静态网站
